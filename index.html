Plural forms in Slavic languages

In Slavic languages, the form is different for each gender and also according to the number of items.
Basically, for every gender three different forms are used according to the number of items:
First for 1 item, second for 2, 3 or 4 items and third for 5+ items, but also for 0 items.
21, 31, ... 101, etc. are using the same form as for 1.
22, 23, 24, 34, 234, etc. are using the same form as for 2, 3 and 4.
You can see examples in Czech language for all three genders below:

  • Masculine:
    N new files сreated
    • Byl vytvořen 1 nový soubor
    • Byly vytvořeny 2 (3, 4) nové soubory
    • Bylo vytvořeno 5 (0, 5+) nových souborů
    • etc...
  • Famine:
    N new folders created
    • Byla vytvořena 1 nová složka
    • Byly vytvořeny 2 (3, 4) nové složky
    • Bylo vytvořeno 5 (0, 5+) nových složek
    • etc...
  • Neutral:
    N new connections created
    • Bylo vytvořeno 1 nové spojení
    • Byla vytvořena 2 (3, 4) nová spojení
    • Bylo vytvořeno 5 (0, 5+) nových spojení
    • etc...
